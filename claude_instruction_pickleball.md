## ミッション
現在、対象のNext.jsアプリケーションにおいて、何らかのエラーや表示崩れ、機能不全が発生しているとの報告（「全然ダメでした」）を受けています。
コードベース全体を走査し、ビルドエラー、ランタイムエラー、コンポーネントの不具合を特定して、すべて自律的に修正してください。

## 特に重点的に確認・修正すべき項目
1. **全体的なエラー修正とデバッグ**
   - サーバーサイド、クライアントサイド双方のエラーログの確認。
   - 依存パッケージの競合や、ビルドプロセス（`npm run build`）が通るかの確認と修正。
2. **AIチャットボットの動作確認と修正**
   - 内蔵されているAIチャットボット機能が正常にマウントされ、ユーザーの入力に対して適切な応答（API連動等）を返せる状態にあるかの確認。
   - エラーで起動しない、UIが崩れている、または応答が返ってこない場合は原因を特定して修正すること。

## アクションプラン
1. 最初にプロジェクトのディレクトリ構成とパッケージ構成（`package.json`）を把握する。
2. 次に、開発サーバー（`npm run dev`）を裏で起動するか、静的解析ツールやビルドコマンドを走らせてエラーの発生箇所を特定する。
3. エラー箇所やAIチャットのソースコード群を特定し、ファイルを修正する。
4. 修正後、再度ビルドやLintチェックを行い、問題が解消されていることを担保してください。

本プロセスはすべて自動承認（無人化）で実行されますので、確認待ちにならずに次々と最適解を適用してプロジェクトを修復してください。期待しています。
